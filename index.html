HTML

  • HTML is a markup language and is used to tell the browser what the structure and content of the webpage are.
  • HTML consists of a variety of different elements, which define how the content behaves.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS allows you to define styles for your webpages. It can be used to style texts, hange colors, font size, etc. as well as create your layout.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch you are currently on.
  • git checkout -b branch name: Creates a new branch and switches to it.

JavaScript

  • JavaScript is a popular programming language.
  • web developers use theconsoleto test code and fix bugs.
  • you can use console.log()to write or log messages to the console.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
